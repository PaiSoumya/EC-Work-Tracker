📚 How to Use Worklog

📋 Tasks Tab - Log Your Work

1

Click "+ Log work" Button

Opens the work entry form to log your tasks and hours.

2

Select a Project

Choose from your existing projects. The form will auto-fill project details like code, manager, and available hours.

3

Enter Work Details

Work Title: Brief description (e.g., "Bug fixes", "Production support")

Work Description: Detailed explanation of what you worked on

Start Date: When you started this work

End Date: When you completed (optional)

Total Hours: Hours spent on this task

4

Daily Hours Breakdown (Optional)

Click "Enter Daily Hours Breakdown" to split hours across multiple days. Useful for tasks spanning several days.

5

Set Task Status

In progress: Currently working on it

Completed: Task is finished

Pending: Waiting to start or blocked

6

Filter & Search

Use the search box and filters to find specific tasks by account, code, status, or time period.

📁 Projects Tab - Manage Projects

1

Click "+ Add Project" Button

Opens the project creation form.

2

Enter Project Information

Account Name: Client or company name

Code: Project identifier or code

Product: Product or service name

Version: Product version (optional)

Manager: Project manager name

Start Date: Project start date

End Date: Expected completion date (optional)

Status: Active, Deactive, or Temporary

Total Hours: Total hours allocated for this project

3

View Project Details

Click the expand icon (▼) to see detailed breakdown of hours claimed, pending hours, and all work entries for that project.

4

Filter Projects

Use "+ Add Filter" to filter by Year, Status, Manager, or Account. Search box helps find specific projects quickly.

5

Understanding Alerts

🔥 Overdue: Project end date has passed

⚠️ High Usage: Over 80% hours claimed

⏰ Ending Soon: Less than 7 days remaining

📊 Analytics Tab - Track Progress

1

Select Time Period

This Week: Current week's data

This Month: Current month's data

Select Month: Choose any specific month

This Quarter: Current quarter (3 months)

This Year: Current year's data

All Time: All your work history

2

View Charts

Weekly Hours Claimed: Bar chart showing hours per week

Project Status Distribution: Pie chart of project statuses

Project Hours Distribution: Bar chart comparing hours across projects

3

Filter by Project

Select a specific project to see analytics for that project only.

4

Understand the Stats

Period: Selected time range

Hours Claimed: Total hours logged in period

Tasks: Number of work entries

Projects: Total active projects

📋 Monday.com Integration

1

Connect Monday.com (Settings Tab)

Go to Settings tab and enter your Monday.com API token and Board ID.

Important: After testing the connection, click "Save Configuration" to enable imports.

2

Import Items

Click "📥 Import Items" to fetch tasks assigned to you from Monday.com.

3

View Imported Items

See all your Monday.com items with their status, hours, and project details.

4

Add to Projects

Click "Add to Project" to convert a Monday.com item into a local project.

⚙️ Settings - Manage Data

1

Export Data

Download all your projects and entries as a JSON file for backup or sharing.

What gets exported:

  • All projects with their details (account, code, manager, hours, etc.)
  • All work entries/tasks with descriptions and hours
  • Monday.com configuration (API token and Board ID)
  • Imported Monday.com items

Use cases:

  • Backup your data before clearing browser cache
  • Share your work tracker setup with team members
  • Transfer data to another computer or browser
  • Keep historical records of your work

Note: The exported file is in JSON format and can be imported back using the "Import Data" feature.

2

Import Data

Load previously exported data from a JSON file.

3

Monday.com Integration

API Token: Get from Monday.com → Profile → Developers → API (either generate a new token or copy existing one)

Board ID: Take the board ID from your Monday.com board URL (the number in the URL)

Important: Click "Test Connection" to verify your credentials, then click "Save Configuration" to enable the import feature.

4

Clear Data

Clear All Data: Removes all projects and entries (cannot be undone!)

Clear Monday.com Data: Removes only imported Monday.com items

💡 Tip: Your data is stored locally in your browser. Each user has their own separate data.

🔒 Privacy: No data is sent to any server. Everything stays on your computer.
